# v0.2.1-pilot

상태: GitHub prerelease 발행 완료
발행일: 2026-07-11

## 변경

`v0.2.0-pilot` downstream validator의 Next telemetry evidence 후보에서 `.env.example` 읽기를
제거한다. AI가 호출하는 bootstrap·validate·security 경로는 파일명이 `.env`로 시작하는 어떤
파일도 읽지 않아야 한다.

## 영향

- Next telemetry 검사는 `package.json`, `scripts/run-next.mjs`, CI의 명시적
  `NEXT_TELEMETRY_DISABLED`만 확인한다.
- `.env`, `.env.local`, `.env.example`, `.environment`와 다른 `.env*` 파일은 읽지 않는다.
- `v0.2.0-pilot`의 다른 정책·adapter·bootstrap·validate 동작은 유지한다.

## Migration

`v0.2.0-pilot`을 적용 중이라면 lock을 갱신하지 말고 `v0.2.1-pilot`로 preview를 다시 만든다.
이미 적용했다면 `scripts/validate-downstream.mjs`를 hotfix version으로 교체하고 전체 검증을 다시
실행한다.

## 검증

- validator source의 `.env.example` 참조 부재 회귀 검사
- downstream validator positive/negative fixture
- `.env*` pre-tool 차단 회귀
- full secret scan·SAST
- clean clone bootstrap·validate
